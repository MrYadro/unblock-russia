# Unblock Russia

Unblock sites that are blocked for Russia because something happened.

Created with OpenWrt, wireguard and addon vpn-policy-routing in mind.

# How to

1. Add `vpn-policy-routing.unblock_russia.ip.user` to your config
1. That's it!
